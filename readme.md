## Setup

To run this project, install it locally using npm or use bat files with prefix install inside each folder:

```
$ This repository contains two parts of WebSite. Each should be started separately. First api, second web.

$ cd ./api
$ npm install

$ cd ./web
$ npm install
```

## How to start

```
$ cd ./api
$ npm run start:dev

$ cd ./web
$ npm start
```
